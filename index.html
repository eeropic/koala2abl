Koala song to Ableton Live set converter v1 alpha

!!! EARLY EARLY ALPHA VERSION, USE AT YOUR OWN RISK !!!
This is a third-party unofficial script, with no affiliation to Elf Audio or Ableton

- Choose a .koala file or drag it below - it will be converted to .ablbundle, that you can open in Ableton Live

- This script uses Ableton Note mobile app's JSON format for the conversion. If you try to open the file on mobile in Ableton Note, it will not work correctly

Feedback: https://github.com/eeropic/koala2abl/issues
